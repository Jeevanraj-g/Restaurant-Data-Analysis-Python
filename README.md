# Restaurant Data Analysis using Python

## Overview
This project analyzes a restaurant dataset using Python and Pandas.

## Technologies Used
- Python
- Pandas
- VS Code

## Tasks
- Read CSV dataset
- Find top cuisines
- Find city with highest number of restaurants
- Calculate average ratings by city
- Perform Exploratory Data Analysis (EDA)

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- GroupBy
- Value Counts
- Aggregation
